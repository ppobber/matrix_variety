# 矩阵的一些基本操作

matrixCal类提供了一些基本的矩阵操作，有：
1）矩阵的加减
2）矩阵的乘和数乘
3）矩阵的转置
4）矩阵的化简（化简为对角矩阵）
5）方阵求逆（初等变换，时间复杂度o(n^3)）
6）方阵的行列式计算（初等变换，时间复杂度o(n^3)）
7）代数余子式法求行列式（时间复杂度较高，但可计算体量较大的矩阵）

由于利用初等变换求解容易产生误差，因此添加了fraction头文件，内包含了numf类用于分数计算。
numf类支持分数的四则运算，包括和数字的四则运算，以及一些必要的符号重载。

matrixCal使用模板，目前没有特化实型，尽量不要使用整型（计算行列式等会产生误差），最好使用numf类。
numf类的模板仅支持整型，推荐使用unsigned long long类型。
